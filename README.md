
## Para empezar

Primero, probá el código en la consola:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Abrí el puerto [http://localhost:3000](http://localhost:3000) en tu navegador para ver los resultados.
